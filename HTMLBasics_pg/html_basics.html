LIST TAGS

Uordered List

Here we have a look at the synatax of the different tags used above and heere as an unorganized list

  • STRONG:"<strong>text</strong>"

    • The strong tag helps provide importance to the taged text by displayign it in bold

  • EMPHASIS:"<em>text</em>"

    • The emphasis tag displays the text in "ITALICS" and gives emphasis to the senternce

  • UORDERED LIST:"<ul>list items</ul>"

    • The unordered list helps display elements in list marked by bulletin points

  • ORDERED LIST:"<ol>list items</ol>"

    • The ordered list displays the elements of he list by numbered points

  • LISTS:"<li>text</ol>"

  • ANCHOR:"<a href="https://www.google.com/">text</a>;

    • The anchor element redirects the user to the link described in the hyperlink reference href
